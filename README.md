# react-todos-app
A HackerNews.com clone using the News API. It include main news feed, search functionality, and "Popular This Week" section.


## About The Project

  A HackerNews.com clone using the News API. It include main news feed, search functionality, and "Popular This Week" section.
  There are three components: searchbar, main news feed, and popular news on the side.

## Built With

- Create-react-app
- Redux
- HTML / CSS

## Prerequisites
 - Git
 - Node.js
 - NPM
